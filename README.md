# Ti82

This is a gem for calculating financial calculator functions (like the
Ti81)

## Installation

Add this line to your application's Gemfile:

    gem 'ti82'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ti82

## Usage

Ti82.pmt()
